## mobbin auth

Before you can use the CLI, you need to log in with your [Mobbin](https://mobbin.com/) account. A free account is sufficient, but some features may be limited depending on your subscription.

### login

Log in to your Mobbin account.

**Interactive mode** (recommended):

```bash
mobbin auth login
```

The CLI will ask for your email and password via secure prompts.

**Non-interactive mode** (for scripts):

```bash
mobbin auth login -e user@example.com -p "votre_mot_de_passe"
```

| Option | Alias | Description |
|--------|-------|-------------|
| `--email` | `-e` | Mobbin account email address |
| `--password` | `-p` | Password |

You can also use the `MOBBIN_EMAIL` and `MOBBIN_PASSWORD` environment variables.

> The session is stored in `~/.config/mobbin/auth.json` with restrictive permissions (600). The token expires after 1 hour but is automatically refreshed.

### logout

Removes the local session.

```bash
mobbin auth logout
```

### status

Checks if you are logged in and displays session information.

```bash
mobbin auth status
```

**Example output:**

```json
{
  "authenticated": true,
  "expired": false,
  "email": "user@example.com",
  "userId": "63585ffe-a240-4615-8b17-afc4bd00629f",
  "expiresAt": "2026-03-15T17:48:46.000Z"
}
```
