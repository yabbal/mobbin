## AppsResource

Explorez les applications indexees par Mobbin.

### search

Recherche d'apps par categorie. Combine le catalogue et les apps populaires pour filtrer.

```typescript
const result = await client.apps.search(
  {
    platform: "ios",
    appCategories: ["Finance", "Shopping"],
  },
  { pageSize: 10 },
);

for (const app of result.data) {
  console.log(`${app.appName} — ${app.appTagline}`);
}
```

### list

Retourne le catalogue complet d'une plateforme. C'est la source de verite pour savoir quelles apps sont disponibles.

```typescript
const allApps = await client.apps.list("ios");
console.log(`${allApps.length} apps iOS disponibles`);

// Filtrer localement
const financeApps = allApps.filter((a) =>
  a.keywords?.includes("finance"),
);
```

### popular

Apps populaires avec previews, organisees par categorie.

```typescript
const popular = await client.apps.popular("ios", 5); // 5 par categorie

for (const app of popular) {
  console.log(`${app.app_name} (${app.app_category}) — score: ${app.popularity_metric}`);
}
```
