## Mobbin for AI agents

The Mobbin CLI and SDK are designed to be used by AI agents. All commands produce **structured JSON** on `stdout`, making them easily parsable by LLMs.

### Why use Mobbin with an AI agent?

When an AI agent (like Claude Code) is working on an interface, it can:

1. **Search for references** — "Show me how Finance apps do their login screen"
2. **Download inspirations** — Get real images from popular apps
3. **Analyze patterns** — Understand which UI patterns are trending
4. **Compare approaches** — See how different apps solve the same UX problem

### JSON output by default

Unlike most CLIs, Mobbin CLI outputs JSON by default (not formatted text). This is a deliberate choice to facilitate integration with agents:

```bash
# The agent can directly parse the output
mobbin screens search --patterns Login -l 3
```
