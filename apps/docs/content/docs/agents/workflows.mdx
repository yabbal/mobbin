## Typical workflows

Here are concrete scenarios where Mobbin enriches work with an AI agent.

---

### 1. Implement a screen from references

You are working on a login screen and want to see how the best apps do it:

```bash
# Step 1: Search for references
mobbin screens search --patterns Login -c "Finance,Shopping" --sort popularity -l 10

# Step 2: Download the best ones
mobbin screens download --patterns Login -c Finance --sort popularity -l 5 -o ./login-refs

# Step 3: The agent can analyze the images and draw inspiration from them
```

---

### 2. UX audit of a competitor

You want to analyze a competitor's app:

```bash
# Find the app
mobbin apps list -p ios | jq '.[] | select(.appName | test("Revolut"))'

# View its screens
mobbin screens search -k "Revolut" -l 20
```

---

### 3. Explore UI trends

What patterns are trending right now?

```bash
# Most trending screens
mobbin screens search --sort trending -l 10

# Trending onboarding patterns
mobbin screens search --patterns Onboarding --sort trending -l 10

# Trending UI elements
mobbin screens search --elements "Bottom Sheet" --sort trending
```

---

### 4. Create a moodboard quickly

```bash
# Download a mix of references
mobbin screens download --patterns "Home" -c Entertainment -l 5 -o ./moodboard
mobbin screens download --patterns "My Account & Profile" -l 5 -o ./moodboard
mobbin screens download --elements "Card" -c "AI" -l 5 -o ./moodboard
```

### 5. Automated export script

```typescript
import { MobbinClient } from "mobbin-sdk";
import { writeFile, mkdir } from "node:fs/promises";

const client = new MobbinClient({
  credentials: {
    email: process.env.MOBBIN_EMAIL!,
    password: process.env.MOBBIN_PASSWORD!,
  },
});
await client.init();

const patterns = ["Login", "Signup", "Checkout", "Settings"];

for (const pattern of patterns) {
  await mkdir(`./export/${pattern}`, { recursive: true });

  const result = await client.screens.search(
    { platform: "ios", screenPatterns: [pattern] },
    { pageSize: 5, sortBy: "popularity" },
  );

  for (const screen of result.data) {
    const buffer = await client.screens.download(
      screen.screenCdnImgSources.src,
    );
    const name = screen.appName.replace(/[^a-zA-Z0-9]/g, "_").toLowerCase();
    await writeFile(
      `./export/${pattern}/${name}.webp`,
      Buffer.from(buffer),
    );
  }

  console.log(`${pattern}: ${result.data.length} screens exported`);
}
```
