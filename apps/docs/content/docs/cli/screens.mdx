## mobbin screens

This is the most powerful command of the CLI. It lets you search through thousands of design screens, download them in high quality, or copy them directly to your clipboard.

---

### search

Search for screens with precise filters.

```bash
mobbin screens search [options]
```

**Practical examples:**

```bash
# All trending login screens on iOS
mobbin screens search --patterns Login

# Onboarding screens in Finance apps, sorted by popularity
mobbin screens search --patterns Onboarding -c Finance --sort popularity

# Screens containing Bottom Sheets and Cards
mobbin screens search --elements "Bottom Sheet,Card"

# Search by visible text on the screen (OCR)
mobbin screens search -k "Forgot password"

# Signup screens on Android
mobbin screens search --patterns Signup -p android

# Animated screens only
mobbin screens search --patterns Home --animation

# Combine multiple filters
mobbin screens search --patterns Checkout -c Shopping -p ios --sort popularity -l 10 -t
```

**Options:**

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--platform` | `-p` | `ios`, `android`, `web` | `ios` |
| `--patterns` | | Screen patterns (comma-separated) | — |
| `--elements` | | UI elements (comma-separated) | — |
| `--keywords` | `-k` | Visible text in screenshots | — |
| `--categories` | `-c` | App categories (comma-separated) | — |
| `--animation` | | Animated screens only | `false` |
| `--sort` | `-s` | `trending`, `publishedAt`, `popularity` | `trending` |
| `--limit` | `-l` | Number of results | `24` |
| `--table` | `-t` | Table display | `false` |

**Popular patterns:** Login, Signup, Home, Checkout, Settings, Notifications, Onboarding, Search, My Account & Profile, Subscription & Paywall, Filter & Sort, Guided Tour & Tutorial.

**Popular UI elements:** Banner, Bottom Sheet, Button, Card, Chip, Dialog, Divider, Stacked List, Tab Bar, Text Field, Toast.

> Use `mobbin filters list -c screenPatterns -t` to see the complete list of 110+ available patterns.

### download

Download high-quality screen images (WebP format).

```bash
mobbin screens download [options]
```

**Examples:**

```bash
# Download 5 signup screens into the current directory
mobbin screens download --patterns Signup -l 5

# Download into a specific folder
mobbin screens download --patterns Checkout -c Shopping -o ./checkout-refs

# Download web references
mobbin screens download -p web --patterns "Landing Page" -l 10 -o ./web-refs
```

Files are named `{app_name}_{id}.webp`. Example: `netflix_fba27ddb.webp`.

**Specific options:**

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--output` | `-o` | Destination directory | `.` (current) |

All `search` options are also available.

**Output:**

```json
{
  "downloaded": [
    {
      "file": "./refs/netflix_fba27ddb.webp",
      "app": "Netflix",
      "patterns": ["Login"]
    }
  ]
}
```

### copy

Copies a screen directly to the clipboard. Perfect for pasting a reference into Figma, Slack, an email, etc.

```bash
mobbin screens copy [options]
```

**Examples:**

```bash
# Copy the first trending login screen
mobbin screens copy --patterns Login

# Copy the 3rd result
mobbin screens copy --patterns Login -i 2

# Copy a paywall screen from Entertainment apps
mobbin screens copy --patterns "Subscription & Paywall" -c Entertainment
```

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--index` | `-i` | Index of the result to copy (0-based) | `0` |

**Compatibility:**

| OS | Method |
|----|--------|
| macOS | `sips` + `osascript` |
| Linux | `xclip` or `wl-copy` (Wayland) |
| Windows | PowerShell `Clipboard` |
