## mobbin filters

Before searching, it is useful to know which filters exist. The `filters` command gives you access to the complete dictionary of patterns, UI elements, and categories.

---

### list

```bash
# View all filter categories
mobbin filters list

# View available screen patterns (110+)
mobbin filters list -c screenPatterns

# View UI elements
mobbin filters list -c screenElements -t

# View app categories
mobbin filters list -c appCategories

# View flow actions
mobbin filters list -c flowActions
```

**Filter categories:**

| Slug | Description | Examples |
|------|-------------|----------|
| `screenPatterns` | Screen types | Login, Signup, Checkout, Settings, Home, Onboarding |
| `screenElements` | UI components | Button, Card, Bottom Sheet, Dialog, Toast, Tab Bar |
| `appCategories` | App categories | Finance, Shopping, Entertainment, Health & Fitness |
| `flowActions` | User actions | Creating Account, Filtering & Sorting, Editing Profile |
| `styles` | Visual styles | Dark Mode, Glassmorphism, Minimalist |

Each dictionary entry contains a `displayName`, a `definition`, and `synonyms`, which helps you understand exactly what each filter represents.

**Tip:** Use filters in combination with `screens search` for very precise results:

```bash
# Paywall screens in Health & Fitness apps with a dark style
mobbin screens search --patterns "Subscription & Paywall" -c "Health & Fitness"
```
