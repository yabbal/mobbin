## ScreensResource

The most important resource of the SDK. It allows you to search through thousands of screens and download them.

### search

Search for screens with precise filters.

```typescript
const result = await client.screens.search(
  {
    platform: "ios",              // "ios" | "android" | "web"
    screenPatterns: ["Login"],    // null to ignore
    screenElements: null,         // ["Button", "Card"] etc.
    screenKeywords: null,         // OCR search
    appCategories: ["Finance"],   // null for all
    hasAnimation: null,           // true for animated screens
  },
  {
    pageSize: 10,                 // Number of results
    sortBy: "trending",           // "trending" | "publishedAt" | "popularity"
  },
);
```

**Response:**

```typescript
result.data // Screen[]
```

Each `Screen` contains:

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique identifier |
| `appName` | `string` | Application name |
| `appCategory` | `string` | App category |
| `screenPatterns` | `string[]` | Detected patterns (Login, Signup...) |
| `screenElements` | `string[]` | Detected UI elements |
| `screenKeywords` | `string[]` | Visible text (OCR) |
| `platform` | `Platform` | ios, android, or web |
| `metadata` | `{ width, height }` | Image dimensions |
| `screenUrl` | `string` | Supabase Storage URL |
| `screenCdnImgSources` | `{ src, srcSet }` | Optimized CDN URLs |

### download

Downloads a screen image via its CDN URL.

```typescript
// Use the CDN src (recommended)
const buffer = await client.screens.download(
  screen.screenCdnImgSources.src,
);

// Save to file
import { writeFile } from "node:fs/promises";
await writeFile("screen.webp", Buffer.from(buffer));
```

Images are in **WebP** format for optimal size with excellent quality.
