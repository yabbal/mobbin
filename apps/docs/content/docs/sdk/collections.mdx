## CollectionsResource

Les collections sont des groupes d'ecrans sauvegardes par l'utilisateur via l'interface web de Mobbin.

### list

```typescript
const collections = await client.collections.list();

for (const collection of collections) {
  console.log(`${collection.name} (${collection.id})`);
}
```
