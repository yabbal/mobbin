## mobbin apps

Explore the applications referenced on Mobbin. Each app contains dozens of captured and annotated screens.

---

### search

Search for apps by category.

```bash
# Finance apps on iOS
mobbin apps search -c Finance

# Shopping and Entertainment apps
mobbin apps search -c "Shopping,Entertainment" -l 10

# Web apps
mobbin apps search -p web -l 5
```

---

### list

Display the complete catalog for a platform. Useful for knowing which apps are available.

```bash
# All iOS apps (800+)
mobbin apps list -p ios

# All web apps
mobbin apps list -p web

# As a table (first 50)
mobbin apps list -t
```

---

### popular

Discover the most popular apps, organized by category.

```bash
# Popular iOS apps
mobbin apps popular

# As a table
mobbin apps popular -t

# Popular web apps, 5 per category
mobbin apps popular -p web -l 5
```
