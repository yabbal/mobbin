## Skill Claude Code

Le fichier `SKILL.md` a la racine du projet permet a Claude Code d'utiliser le CLI Mobbin automatiquement quand vous parlez de design, de references UI ou de patterns d'ecrans.

### Installation

Le skill est actif automatiquement si le fichier `SKILL.md` est present dans votre projet. Il suffit de copier le fichier depuis le repo :

```bash
curl -o SKILL.md https://raw.githubusercontent.com/yabbal/mobbin/main/SKILL.md
```

### Utilisation

Une fois le skill installe, vous pouvez demander a Claude Code :

- *"Montre-moi des references de login pour une app Finance"*
- *"Telecharge 5 ecrans de checkout dans ./refs"*
- *"Quels patterns d'ecran existent sur Mobbin ?"*
- *"Copie un ecran de paywall dans mon presse-papier"*
- *"Quelles sont les apps les plus populaires sur iOS ?"*

Claude Code utilisera automatiquement les bonnes commandes CLI.

### Prerequis

Le CLI doit etre installe et authentifie :

```bash
pnpm add -g mobbin-cli
mobbin auth login
```
