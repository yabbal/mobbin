## mobbin-cli

Le CLI Mobbin transforme votre terminal en outil de recherche de design. Il se connecte a l'API Mobbin pour vous donner acces a des milliers de references UI sans quitter votre workflow.

### Installation

```bash
pnpm add -g mobbin-cli
```

Apres installation, la commande `mobbin` est disponible globalement.

### Premiere utilisation

```bash
# Verifier l'installation
mobbin version

# Se connecter
mobbin auth login

# Lancer votre premiere recherche
mobbin screens search --patterns Login -l 5
```

### Commandes disponibles

| Commande | Description |
|----------|-------------|
| [`mobbin auth`](/docs/cli/auth) | Se connecter, se deconnecter, verifier le statut |
| [`mobbin screens`](/docs/cli/screens) | Rechercher, telecharger et copier des ecrans |
| [`mobbin apps`](/docs/cli/apps) | Explorer le catalogue d'applications |
| [`mobbin filters`](/docs/cli/filters) | Decouvrir les filtres disponibles |
| [`mobbin collections`](/docs/cli/collections) | Gerer vos collections sauvegardees |
| `mobbin version` | Afficher la version du CLI |

### Format de sortie

Par defaut, toutes les commandes produisent du **JSON** sur `stdout`. C'est ideal pour :

- Le piping vers `jq` ou d'autres outils
- L'integration avec des agents IA (Claude Code, etc.)
- L'automatisation dans des scripts

Pour un affichage humain en tableau, ajoutez `--table` ou `-t` :

```bash
# JSON (defaut)
mobbin apps popular | jq '.[0:3] | .[].app_name'

# Tableau
mobbin apps popular -t
```

### Filtres globaux

La plupart des commandes de recherche partagent ces options :

| Option | Alias | Description | Valeurs |
|--------|-------|-------------|---------|
| `--platform` | `-p` | Plateforme cible | `ios` (defaut), `android`, `web` |
| `--sort` | `-s` | Tri des resultats | `trending` (defaut), `publishedAt`, `popularity` |
| `--limit` | `-l` | Nombre de resultats | Nombre (defaut: 24) |
| `--table` | `-t` | Affichage en tableau | Flag |
