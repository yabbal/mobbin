## mobbin apps

Explorez les applications referencees sur Mobbin. Chaque app contient des dizaines d'ecrans captures et annotes.

---

### search

Recherchez des apps par categorie.

```bash
# Apps Finance sur iOS
mobbin apps search -c Finance

# Apps Shopping et Entertainment
mobbin apps search -c "Shopping,Entertainment" -l 10

# Apps web
mobbin apps search -p web -l 5
```

---

### list

Affichez le catalogue complet d'une plateforme. Utile pour savoir quelles apps sont disponibles.

```bash
# Toutes les apps iOS (800+)
mobbin apps list -p ios

# Toutes les apps web
mobbin apps list -p web

# En tableau (50 premiers)
mobbin apps list -t
```

---

### popular

Decouvrez les apps les plus populaires, organisees par categorie.

```bash
# Apps populaires iOS
mobbin apps popular

# En tableau
mobbin apps popular -t

# Apps populaires web, 5 par categorie
mobbin apps popular -p web -l 5
```

**Exemple de sortie (tableau) :**

```

```
