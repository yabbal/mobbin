## What is Mobbin CLI & SDK?

[Mobbin](https://mobbin.com/) is a reference platform for UI/UX design. It catalogs thousands of mobile and web application screens, organized by patterns, UI elements, and categories.

**Mobbin CLI & SDK** are unofficial tools that allow you to access all this content **directly from your terminal or TypeScript code**, without using a browser.

### Who is this tool for?

- **Designers** who want to quickly download references for a project
- **Developers** who are implementing interfaces and looking for inspiration
- **AI Agents** (Claude Code, etc.) that need to access visual references programmatically
- **Product teams** who want to analyze the UI patterns of their competitors

### What you can do

- **Search** screens by pattern (Login, Signup, Checkout, Onboarding...), by UI element (Button, Card, Dialog...), by app category (Finance, Shopping...) or by visible text in screenshots (OCR)
- **Download** high-quality screen images
- **Copy** a screen directly to your clipboard
- **Explore** the catalog of 800+ apps on iOS, Android, and Web
- **List** available filters to refine your searches

## Installation

**CLI** — for command-line usage:

```bash
pnpm add -g mobbin-cli
```

**SDK** — for usage in TypeScript code:

```bash
pnpm add mobbin-sdk
```

## Quick start

### 1. Log in to your Mobbin account

```bash
mobbin auth login
```

The CLI will ask for your email and password. The session is stored locally in `~/.config/mobbin/auth.json`.

### 2. Search for screens

```bash
# Login screens from iOS apps
mobbin screens search --patterns Login

# Checkout screens in Shopping apps, sorted by popularity
mobbin screens search --patterns Checkout -c Shopping --sort popularity

# Screens with Bottom Sheets
mobbin screens search --elements "Bottom Sheet" -l 10
```

### 3. Download references

```bash
# Download 5 signup screens into a folder
mobbin screens download --patterns Signup -l 5 -o ./design-refs
```

### 4. Copy to clipboard

```bash
# Copy the first result directly
mobbin screens copy --patterns "Subscription & Paywall"
```

## Two packages, one ecosystem

| Package | Description | When to use |
|---------|-------------|-------------|
| [**mobbin-cli**](/docs/cli) | Command-line interface | Quick search, download, script integration |
| [**mobbin-sdk**](/docs/sdk) | TypeScript SDK | Integration in your applications, AI agents, automation |

The CLI uses the SDK internally. Both packages are published on npm with OIDC provenance.
