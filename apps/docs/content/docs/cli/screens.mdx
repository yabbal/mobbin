## mobbin screens

C'est la commande la plus puissante du CLI. Elle vous permet de chercher parmi des milliers d'ecrans de design, de les telecharger en haute qualite ou de les copier directement dans votre presse-papier.

---

### search

Recherchez des ecrans avec des filtres precis.

```bash
mobbin screens search [options]
```

**Exemples concrets :**

```bash
# Tous les ecrans de login tendance sur iOS
mobbin screens search --patterns Login

# Ecrans d'onboarding dans les apps Finance, tries par popularite
mobbin screens search --patterns Onboarding -c Finance --sort popularity

# Ecrans contenant des Bottom Sheets et des Cards
mobbin screens search --elements "Bottom Sheet,Card"

# Recherche par texte visible dans l'ecran (OCR)
mobbin screens search -k "Forgot password"

# Ecrans de signup sur Android
mobbin screens search --patterns Signup -p android

# Ecrans animes uniquement
mobbin screens search --patterns Home --animation

# Combiner plusieurs filtres
mobbin screens search --patterns Checkout -c Shopping -p ios --sort popularity -l 10 -t
```

**Options :**

| Option | Alias | Description | Defaut |
|--------|-------|-------------|--------|
| `--platform` | `-p` | `ios`, `android`, `web` | `ios` |
| `--patterns` | | Patterns d'ecran (virgules) | — |
| `--elements` | | Elements UI (virgules) | — |
| `--keywords` | `-k` | Texte visible dans les screenshots | — |
| `--categories` | `-c` | Categories d'app (virgules) | — |
| `--animation` | | Ecrans animes uniquement | `false` |
| `--sort` | `-s` | `trending`, `publishedAt`, `popularity` | `trending` |
| `--limit` | `-l` | Nombre de resultats | `24` |
| `--table` | `-t` | Affichage en tableau | `false` |

**Patterns populaires :** Login, Signup, Home, Checkout, Settings, Notifications, Onboarding, Search, My Account & Profile, Subscription & Paywall, Filter & Sort, Guided Tour & Tutorial.

**Elements UI populaires :** Banner, Bottom Sheet, Button, Card, Chip, Dialog, Divider, Stacked List, Tab Bar, Text Field, Toast.

> Utilisez `mobbin filters list -c screenPatterns -t` pour voir la liste complete des 110+ patterns disponibles.

### download

Telechargez des images d'ecrans en haute qualite (format WebP).

```bash
mobbin screens download [options]
```

**Exemples :**

```bash
# Telecharger 5 ecrans de signup dans le dossier courant
mobbin screens download --patterns Signup -l 5

# Telecharger dans un dossier specifique
mobbin screens download --patterns Checkout -c Shopping -o ./checkout-refs

# Telecharger des references pour le web
mobbin screens download -p web --patterns "Landing Page" -l 10 -o ./web-refs
```

Les fichiers sont nommes `{app_name}_{id}.webp`. Exemple : `netflix_fba27ddb.webp`.

**Options specifiques :**

| Option | Alias | Description | Defaut |
|--------|-------|-------------|--------|
| `--output` | `-o` | Repertoire de destination | `.` (courant) |

Toutes les options de `search` sont aussi disponibles.

**Sortie :**

```json
{
  "downloaded": [
    {
      "file": "./refs/netflix_fba27ddb.webp",
      "app": "Netflix",
      "patterns": ["Login"]
    }
  ]
}
```

### copy

Copie un ecran directement dans le presse-papier. Parfait pour coller une reference dans Figma, Slack, un email, etc.

```bash
mobbin screens copy [options]
```

**Exemples :**

```bash
# Copier le premier ecran de login tendance
mobbin screens copy --patterns Login

# Copier le 3e resultat
mobbin screens copy --patterns Login -i 2

# Copier un ecran de paywall dans les apps Entertainment
mobbin screens copy --patterns "Subscription & Paywall" -c Entertainment
```

| Option | Alias | Description | Defaut |
|--------|-------|-------------|--------|
| `--index` | `-i` | Index du resultat a copier (0-based) | `0` |

**Compatibilite :**

| OS | Methode |
|----|---------|
| macOS | `sips` + `osascript` |
| Linux | `xclip` ou `wl-copy` (Wayland) |
| Windows | PowerShell `Clipboard` |
