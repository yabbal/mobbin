## mobbin collections

Les collections sont des groupes d'ecrans que vous avez sauvegardes sur Mobbin via l'interface web. Le CLI vous permet d'y acceder.

### list

```bash
mobbin collections list
```

Retourne un tableau JSON de vos collections avec leur nom et identifiant.
