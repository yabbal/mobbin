## FiltersResource

The filter dictionary contains all possible values for searches. This is useful for building search interfaces or discovering available options.

### list

```typescript
const categories = await client.filters.list();

for (const cat of categories) {
  console.log(`\n${cat.displayName} (${cat.slug}):`);
  for (const sub of cat.subCategories) {
    const names = sub.entries.map((e) => e.displayName);
    console.log(`  ${sub.displayName}: ${names.join(", ")}`);
  }
}
```

**Response structure:**

```typescript
interface DictionaryCategory {
  slug: string;           // "screenPatterns", "screenElements", etc.
  displayName: string;    // Human-readable name
  experience: "web" | "mobile";
  subCategories: {
    displayName: string;
    entries: {
      displayName: string; // E.g.: "Login"
      definition: string;  // Pattern description
      synonyms: string[];  // Alternative terms
    }[];
  }[];
}
```

Each entry has a `definition` that explains the pattern, and `synonyms` to help with search.
