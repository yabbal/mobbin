## Mobbin CLI & SDK

Outils non officiels pour interagir avec [Mobbin](https://mobbin.com/) depuis le terminal ou depuis du code TypeScript.

### Installation

**CLI (global) :**

```bash
pnpm add -g mobbin-cli
```

**SDK (dans un projet) :**

```bash
pnpm add mobbin-sdk
```

### Demarrage rapide

```bash
# Se connecter
mobbin auth login

# Rechercher des ecrans de login
mobbin screens search --patterns Login

# Telecharger des references de design
mobbin screens download --patterns Signup -l 5 -o ./refs

# Copier un ecran dans le presse-papier
mobbin screens copy --patterns Checkout
```

### Plateformes supportees

- **iOS** — Applications mobiles iPhone/iPad
- **Android** — Applications mobiles Android
- **Web** — Sites et applications web
