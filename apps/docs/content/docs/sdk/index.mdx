## mobbin-sdk

The TypeScript SDK allows you to integrate Mobbin into your applications, scripts, or AI agents. It is designed to be lightweight and easy to use.

### Features

- **Zero runtime dependencies** — uses native `fetch` (Node.js 18+)
- **TypeScript first** — complete types for all API responses
- **Automatic session management** — automatic token refresh
- **Auth via Supabase cookies** — compatible with the Mobbin API

### Installation

```bash
pnpm add mobbin-sdk
```

### Initialization

```typescript
import { MobbinClient } from "mobbin-sdk";

// Option 1: Login with credentials
const client = new MobbinClient({
  credentials: { email: "user@example.com", password: "..." },
});
await client.init(); // Logs in automatically

// Option 2: Existing session (from a file, a store, etc.)
const client = new MobbinClient({ session: savedSession });
await client.init();

// Option 3: Client without auth (for public endpoints)
const client = new MobbinClient();
```

### Available resources

| Resource | Methods | Description |
|----------|---------|-------------|
| [`client.auth`](/docs/sdk/auth) | `checkEmail`, `login`, `refresh`, `getUser` | Authentication |
| [`client.screens`](/docs/sdk/screens) | `search`, `download` | Design screens |
| [`client.apps`](/docs/sdk/apps) | `search`, `list`, `popular` | App catalog |
| [`client.filters`](/docs/sdk/filters) | `list` | Filter dictionary |
| [`client.collections`](/docs/sdk/collections) | `list` | User collections |

### Main types

```typescript
import type {
  Platform,        // "ios" | "android" | "web"
  SortBy,          // "trending" | "publishedAt" | "popularity"
  Screen,          // Screen with metadata, patterns, elements, CDN URLs
  SearchableApp,   // App from the catalog
  PopularApp,      // Popular app
  DictionaryCategory, // Filter category
  SupabaseSession, // Authentication session
} from "mobbin-sdk";
```

### Complete example

```typescript
import { MobbinClient } from "mobbin-sdk";
import { writeFile } from "node:fs/promises";

const client = new MobbinClient({
  credentials: { email: "user@example.com", password: "..." },
});
await client.init();

// Search for checkout screens in Finance apps
const result = await client.screens.search(
  {
    platform: "ios",
    screenPatterns: ["Checkout"],
    appCategories: ["Finance"],
  },
  { pageSize: 5, sortBy: "popularity" },
);

console.log(`${result.data.length} screens found`);

// Download the first screen
const screen = result.data[0];
const buffer = await client.screens.download(screen.screenCdnImgSources.src);
await writeFile(`${screen.appName}_checkout.webp`, Buffer.from(buffer));

console.log(`Downloaded: ${screen.appName} (${screen.screenPatterns.join(", ")})`);
```
