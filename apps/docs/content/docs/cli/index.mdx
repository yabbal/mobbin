## mobbin-cli

The Mobbin CLI turns your terminal into a design search tool. It connects to the Mobbin API to give you access to thousands of UI references without leaving your workflow.

### Installation

```bash
pnpm add -g mobbin-cli
```

After installation, the `mobbin` command is available globally.

### First use

```bash
# Verify the installation
mobbin version

# Log in
mobbin auth login

# Run your first search
mobbin screens search --patterns Login -l 5
```

### Available commands

| Command | Description |
|---------|-------------|
| [`mobbin auth`](/docs/cli/auth) | Log in, log out, check status |
| [`mobbin screens`](/docs/cli/screens) | Search, download, and copy screens |
| [`mobbin apps`](/docs/cli/apps) | Explore the application catalog |
| [`mobbin filters`](/docs/cli/filters) | Discover available filters |
| [`mobbin collections`](/docs/cli/collections) | Manage your saved collections |
| `mobbin version` | Display the CLI version |

### Output format

By default, all commands output **JSON** to `stdout`. This is ideal for:

- Piping to `jq` or other tools
- Integration with AI agents (Claude Code, etc.)
- Automation in scripts

For a human-readable table display, add `--table` or `-t`:

```bash
# JSON (default)
mobbin apps popular | jq '.[0:3] | .[].app_name'

# Table
mobbin apps popular -t
```

### Global filters

Most search commands share these options:

| Option | Alias | Description | Values |
|--------|-------|-------------|--------|
| `--platform` | `-p` | Target platform | `ios` (default), `android`, `web` |
| `--sort` | `-s` | Sort results | `trending` (default), `publishedAt`, `popularity` |
| `--limit` | `-l` | Number of results | Number (default: 24) |
| `--table` | `-t` | Table display | Flag |
