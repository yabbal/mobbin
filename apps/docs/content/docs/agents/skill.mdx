## Claude Code Skill

The `SKILL.md` file at the project root allows Claude Code to automatically use the Mobbin CLI when you talk about design, UI references, or screen patterns.

### Installation

The skill is automatically active if the `SKILL.md` file is present in your project. Simply copy the file from the repo:

```bash
curl -o SKILL.md https://raw.githubusercontent.com/yabbal/mobbin/main/SKILL.md
```

### Usage

Once the skill is installed, you can ask Claude Code:

- *"Show me login references for a Finance app"*
- *"Download 5 checkout screens into ./refs"*
- *"What screen patterns exist on Mobbin?"*
- *"Copy a paywall screen to my clipboard"*
- *"What are the most popular apps on iOS?"*

Claude Code will automatically use the right CLI commands.

### Prerequisites

The CLI must be installed and authenticated:

```bash
pnpm add -g mobbin-cli
mobbin auth login
```
