## AppsResource

Explore the applications indexed by Mobbin.

### search

Search for apps by category. Combines the catalog and popular apps for filtering.

```typescript
const result = await client.apps.search(
  {
    platform: "ios",
    appCategories: ["Finance", "Shopping"],
  },
  { pageSize: 10 },
);

for (const app of result.data) {
  console.log(`${app.appName} — ${app.appTagline}`);
}
```

### list

Returns the complete catalog for a platform. This is the source of truth for knowing which apps are available.

```typescript
const allApps = await client.apps.list("ios");
console.log(`${allApps.length} iOS apps available`);

// Filter locally
const financeApps = allApps.filter((a) =>
  a.keywords?.includes("finance"),
);
```

### popular

Popular apps with previews, organized by category.

```typescript
const popular = await client.apps.popular("ios", 5); // 5 per category

for (const app of popular) {
  console.log(`${app.app_name} (${app.app_category}) — score: ${app.popularity_metric}`);
}
```
